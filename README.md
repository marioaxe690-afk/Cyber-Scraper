# 🚀 Cyber-Scraper (全能赛博情报中心)

> 这是一个硬核的自动化数据收割与情报分析终端。
> 从前端 DOM 深度解析，到绕过系统底层代理 Bug；从多线程提速，到生成全自动排版的工业级 Excel 报表，全栈拿捏。

## ⚔️ 核心火力 (Features)

本情报中心目前已实装三大核心收割模块：
* 🎓 **[1] 学术前沿雷达**：穿透海外代理，直捣 arXiv 核心，精准提取最新 CS 顶级前沿论文及超长摘要。
* 📚 **[2] 高分知识收割**：国内网络极速直连，秒级荡平豆瓣 Top250，构建认知提升书单。
* 🎮 **[3] 赛博娱乐监控**：精准对接 Clash 隐秘通道 (Port: 7897)，实时抓取 Steam 骨折特惠游戏情报。

## 🛠️ 技术栈 (Tech Stack)
* **网络穿透**：`requests`, 自定义本地/全局代理分流策略, 免疫 SSL 证书拦截。
* **DOM 解析**：`BeautifulSoup4`, 强力无视标签嵌套的文本榨取引擎。
* **数据工程**：`pandas` 数据清洗 + `openpyxl` 自动化 Excel 报表美化排版。
* **程序封装**：`pyinstaller` 一键封装为独立终端执行程序 (.exe)。
